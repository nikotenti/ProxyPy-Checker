## About
This tool get proxies from the network or local file and checks connection status.

## Installation
### Dependencies
- Python 3.9
- beautifulsoup4==4.9.3
- pyfiglet==0.8.post1
- requests==2.25.1
- termcolor==1.1.0
```
$ pip3 install -r requirements.txt
```

### Usage
```
$ python3.9 main.py
```
